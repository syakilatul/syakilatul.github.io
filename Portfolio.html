Portfolio Saya

Berikut adalah beberapa project yang pernah saya kerjakan. Setiap project menjadi pengalaman berharga untuk mengasah keterampilan saya dalam pengembangan website, baik dari sisi desain antarmuka maupun fungsionalitasnya.

Aplikasi Absensi

Aplikasi absensi berbasis web yang dibuat untuk mencatat kehadiran siswa atau pegawai. Sistem ini memiliki fitur login sederhana, input data kehadiran, serta rekap laporan. Project ini membantu saya memahami konsep CRUD (Create, Read, Update, Delete) dengan PHP dan MySQL, sekaligus mengasah kemampuan desain tampilan agar mudah digunakan.

Website Musik

Website streaming musik sederhana di mana pengguna bisa memutar lagu dan melihat daftar musik favorit. Dibangun dengan HTML, CSS, dan JavaScript, project ini fokus pada desain UI/UX yang menarik dengan pemutar audio interaktif. Dari project ini, saya belajar lebih dalam tentang manipulasi DOM, penggunaan event listener, dan integrasi file audio.

To-do List Web App

Aplikasi to-do list berbasis web untuk mencatat tugas sehari-hari. Pengguna dapat menambah, mengedit, dan menghapus daftar kegiatan mereka. Project ini mengajarkan saya cara mengatur data secara dinamis menggunakan JavaScript, serta bagaimana menyimpan data sementara dengan Local Storage agar data tidak hilang meskipun halaman direfresh.
